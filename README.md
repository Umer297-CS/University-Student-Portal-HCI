# University Student Portal (HCI)

## Overview

The **University Student Portal (USP)** is a Human-Computer Interaction (HCI) project that focuses on designing a user-friendly and interactive academic portal for university students. The interface was designed and prototyped in **Figma**, following User-Centered Design (UCD) principles to provide an intuitive, accessible, and consistent user experience.

## Features

* Secure Login Interface
* Student Dashboard
* Course Management
* Weekly Timetable
* Attendance Tracking
* Grade Management
* Announcements
* Course Enrollment
* Fee Management
* Student Profile
* Logout Functionality
* Interactive Navigation
